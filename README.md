gcs-designer 是一个基于 Vue 3 + Meta2d.js 的工业组态可视化编辑器，支持 Web 和 Electron 桌面双平台。核心功能包括：
图形编辑 - 拖拽式画布编辑，支持流程图、时序图、类图、表单图等多种图表类型
图元库 - 内置废气处理、轨道交通等工业 SVG 图元，支持自定义图标
属性配置 - 外观样式、动画效果、事件响应、数据绑定等完整配置面板
实时预览 - 一键预览组态效果，支持数据驱动渲染
桌面化 - Electron 打包为 Windows/Mac 桌面应用，支持本地文件读写
🎯 核心痛点解决
解决工业监控系统的可视化开发门槛问题：
传统组态软件昂贵且封闭，本项目提供开源免费的 Web 端替代方案
非技术人员通过拖拽即可完成监控界面搭建，无需编码
支持实时数据绑定，实现 IoT 设备状态的可视化监控

用户操作 → 事件总线(emitter) → 状态更新 → 画布渲染
画布初始化 - Meta2d 实例挂载，注册各类图元插件
图元操作 - 拖拽/选中/缩放触发 active/inactive 事件
属性同步 - 右侧面板通过 meta2d.setValue() 双向绑定图元属性
数据驱动 - 通过 meta2d.on('valueUpdate') 监听数据变化，实时重绘画布
消息处理 - 外部数据通过 messageHandler 推送到绑定字段，驱动 UI 更新
